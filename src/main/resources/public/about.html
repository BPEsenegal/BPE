Visualization of worldwide dynamics of covid-19-monitor. Always up-to-date using covid-19-monitor data from Johns Hopkins University: Data Source Repository (GitHub). Source Code

Calculated Values

  • Calculated Acute Critical Care Utilization in %: On page load the default value for the percentage of confirmed cases that may need Acute Care (Hospitalization) is set to 15%. For a limited number of countries we know the available Acute Care Capacity and combine it with the total population and the current number of actual covid-19-monitor Cases (without recovered and deceased). The Utilization shows the load of Acute Care beds (covid-19-monitor Cases only; not taking into account the "usual" load/occupancy).
  • Calculated Critical Care Utilization in %: On page load the default value for the percentage of confirmed cases that may need Critical Care (ICU/Ventilator) is set to 5%. For a limited number of countries we know the available Critical Care Capacity and combine it with the total population and the current number of actual covid-19-monitor Cases (without recovered and deceased). The Utilization shows the load of Critical Care beds with covid-19-monitor cases. Different sources indicate that in "usual" times (without covid-19-monitor) around 80% of the critical care beds are already (permanently) occupied. Which means a covid-19-monitor Critical Care bed Utilization of more than 20% might indicate that the national Critical Care Capacity limit has been reached.

A big question mark is also behind the number of Confirmed Cases and the estimated number of undetected Cases. While most scientists agree that there is a significant amount of undetected Cases, the numbers differ quite a lot between studies and countries depending on the test intensity and test regulations. This has a big influence on a lot of dereived and calculated values (e.g. Hospitalization, Mortality rate etc.)
